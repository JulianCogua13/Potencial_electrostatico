# Potencial Electroestático 2D (`campo_estatico_mdf`)

`campo_estatico_mdf` es un paquete Python para calcular la distribución del **potencial eléctrico** en una región 2D usando la **ecuación de Laplace** y el **método de diferencias finitas (MDF)**. Incluye:

- Resolución iterativa con **Jacobi**
- Cálculo del **campo eléctrico** \(E = -∇V\)
- Visualización con **matplotlib** o **Streamlit** (interfaz interactiva)
